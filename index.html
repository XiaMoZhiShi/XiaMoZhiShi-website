关于我们

因梦而生

夏沫之诗是一群有着共同梦想的 Minecraft 玩家创建的社区，我们希望以我们的微薄之力为整个 Minecraft 社区做出贡献
我们遵循开源精神，将服务器玩法全部开源在 GitHub 上，这也是我们社区的精神

敬畏生命

服务器本周目的核心精神为敬畏生命，受 Re:Zero 的启发，我们创造了我们独特的重生机制，以有限的生命时时刻刻提醒人们敬爱生命。

团结就是力量

我们倡导团结、互助的社区精神，在服务器内践行于此。我们致力于将服务器打造成人类命运共同体的典范，践行新时代社会主义精神。图为夏沫之诗第一周目关服纪念合影

共同应对灾难
