因梦而生

夏沫之诗是一群有着共同梦想的 Minecraft 玩家创建的社区，我们希望以我们的微薄之力为整个 Minecraft 社区做出贡献
我们遵循开源精神，将服务器玩法全部开源在 GitHub 上，这也是我们社区的精神

伪装机制

服务器自研的FeatherMorph 伪装插件可以使玩家伪装为已击杀的生物甚至玩家，并获得相应生物技能，为传统生存带来新的趣味与挑战。

团结就是力量

我们践行并倡导团结、合作的精神。我们致力于将服务器打造成人类命运共同体的典范，践行开放协作的社区精神。

共同应对灾难
